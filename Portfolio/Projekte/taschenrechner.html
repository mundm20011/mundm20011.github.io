Python Taschenrechner

Programmieren Sie einen einfachen Taschenrechner mit Python-Funktionen!

Dieses Projekt ist eine einfache Anwendung zum Programmieren eines Taschenrechners mit Python-Funktionen. Der Taschenrechner soll in der Lage sein einfache Additions-, Subtraktions-, Multiplikations- und Division-Operationen durchzuführen und die Ergebnisse im Terminal anzuzeigen.

Funktionen

Der Taschenrechner soll folgende Funktionen enthalten:

- `add()` : Diese Funktion soll zwei Zahlen addieren und das Ergebnis im Terminal anzeigen.
- `subtract()` : Diese Funktion soll zwei Zahlen subtrahieren und das Ergebnis im Terminal anzeigen.
- `multiply()` : Diese Funktion soll zwei Zahlen multiplizieren und das Ergebnis im Terminal anzeigen.
- `divide()` : Diese Funktion soll zwei Zahlen dividieren und das Ergebnis im Terminal anzeigen.

Anforderungen

- Programmieren Sie den Taschenrechner mit Python-Funktionen.
- Die Ergebnisse sollen im Terminal angezeigt werden.
- Die Funktionen `add()`, `subtract()`, `multiply()` und `divide()` sollen mittels Funktionen programmiert werden.

Bonusaufgabe

- Fügen Sie eine Funktion hinzu, mit der man Quadratwurzeln berechnen kann.
- Fügen Sie eine Funktion hinzu, mit der man Palindrome (Wörter oder Sätze, die vorwärts und rückwärts gelesen das gleiche sind) überprüfen kann. (Beispiel: "Anna" ist ein Palindrom, da es vorwärts und rückwärts gelesen das gleiche ist.)
- Fügen Sie eine Funktion hinzu, mit der man Primzahlen überprüfen kann.
- Fügen Sie eine Funktion hinzu, mit der man die Fakultät einer Zahl berechnen kann.
- Fügen Sie eine Funktion hinzu, mit der man die Summe einer Zahlenfolge berechnen kann.
- Fügen Sie eine Funktion hinzu, mit der man binäre Zahlen in Dezimalzahlen umwandeln kann.
